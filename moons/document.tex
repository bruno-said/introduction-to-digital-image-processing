\section{Discussão Técnica}
Podemos aplicar filtro as imagens tanto no domínio do espaço como no domínio da 
frequência. Além disso, podemos criar a mesma filtragem no domínio do espaço no
domínio da frequência usando a Tranformada de Fourier. Nesse projeto vamos criar
as menas images que serião criadas pelos filtro Laplaciano espacial. Para
realizar esse operação nos pricedemos da seguinte maneira.

\begin{itemize}
  \item Obter a imagem M x N que se deseja aplicar o filtro.
  \item Expandir a imagem para P x Q. Geralmente P = 2M e Q = 2N.
  \item Aplicar a Transformada de Fourier a imagem expandidad.
  \item Expandir o filtro no domínio do espaço para a mesma dimenção P x Q.
  \item Aplicar a Tranformada de Fourier ao filtro expandido.
  \item Multiplicar os dois filtro pela multiplicação de Hadamard, ou seja, 
  item por item.
  \item Aplicar a Transformada Inversa no resultado obtido
  \item Recordar a imagem M x N
  \item Resultado final encotrado
\end{itemize}

\section{Conclusões}
Podemos concluir que as imagens finais são identicas as geradas pelo filtro no
domínio espacial. Sendo assim, uma vantegem da troca de domínios é a velocidade
no precessamento da imagem pois qualquer um dos filtros pode ser obtido nos em
qualquer um dos dois domínios.
